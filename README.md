# FrameStory 웹사이트

브랜드 스토리를 판매로 연결하는 디지털 프레임워크를 제공하는 FrameStory의 공식 웹사이트입니다.

## 프로젝트 개요

FrameStory는 단순한 웹사이트 제작을 넘어 브랜드 내러티브를 기반으로 한 종합적 판매 전략 시스템을 구축합니다.

### 주요 기능

1. **3가지 프레임워크 소개**
   - StoryFrame: 브랜드 스토리 개발
   - StrategyFrame: 마케팅 전략 수립
   - WebFrame: 웹사이트 개발 및 최적화

2. **직관적 프로젝트 시작 시스템**
   - 5단계 간단 설문으로 맞춤형 제안서 제공
   - 비즈니스 유형별 맞춤 솔루션 제안

3. **모바일 최적화**
   - 모든 기기에서 최적화된 사용자 경험
   - 반응형 디자인 구현
